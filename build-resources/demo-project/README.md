# Flint Demo Project

This project contains intentional design system violations to demonstrate what Flint detects.

## What you'll see
- Color drift: hardcoded hex values that differ from design tokens
- Typography violations: wrong font sizes and weights
- Accessibility issues: missing aria labels and contrast failures

## What to try
1. Open DemoCard.tsx in Glass to see violations highlighted
2. Say "fix it" in your AI assistant to watch Flint auto-remediate
3. Check the governance panel for the health score

## Note
These violations are intentional. This is a teaching demo.
