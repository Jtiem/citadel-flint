Phase 11: Canvas Views & Build Mode

10 min

Canvas View Toggle

Top-center of the canvas: a 3-segment toggle — Preview, Build, Govern
Clicking each segment switches the canvas view (layout changes)
Preview mode shows the LivePreview iframe (existing behavior)

Build Mode

Switch to Build — canvas shows component cards (not the iframe)
Component cards show: thumbnail, component name, category badge
Cards have an "Insert" button
Clicking a card selects it (indigo ring highlight)

Recipe Strip

In Build mode: a horizontally-scrollable strip of recipe chips at the top
Recipes include: Login Form, Nav Bar, Data Table, Notification, etc.
Each chip shows a green or amber dot (green = all components available, amber = some missing)
Clicking a recipe inserts it into the active file

Govern Mode

Switch to Govern — cards show health grade, ΔE score, A11y status
Dependency edges connect cards that import each other
What to look for: Is the toggle obvious? Does Build mode feel like a useful discovery surface, or just a grid of cards? Are the recipe chips self-explanatory?
