# Demo 03 — Mithril Shadow Audit (Perceptual Drift Detection)

`drift-component.tsx` is a pricing card whose author eyeballed color values from a Figma screenshot rather than reading token values directly — the results look plausible at a glance but are measurably wrong. The Mithril linter converts every color to CIELAB color space and computes CIEDE2000 ΔE (perceptual distance) between the hardcoded value and the nearest design token: a ΔE below 2.0 is imperceptible to human vision and passes; above 2.0 triggers an amber warning; above 10.0 triggers a critical violation. This component contains five violations — the most severe being `#FF3333` (bright red badge, ΔE ≈ 58 from `color.primary`) and `#00AAFF` (sky-blue icon, ΔE ≈ 19) — plus a 1px font-size drift (`15px` vs. the `14px` token) that would be invisible in a code review but is caught deterministically by `MITHRIL-TYP-002`.
