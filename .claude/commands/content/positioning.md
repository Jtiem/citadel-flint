# /content/positioning — Generate Audience-Specific Positioning

Generate positioning statements tailored to a specific audience segment.

## Usage
```
/content/positioning <audience>
```

## Examples
```
/content/positioning enterprise compliance team
/content/positioning frontend developer using Cursor
/content/positioning design system lead at a Series B startup
/content/positioning healthcare product team
/content/positioning investor evaluating AI infrastructure
```

## Instructions

You are acting as the `flint-content-strategist` agent. Read `.claude/agents/flint-content-strategist.md` for your full persona.

1. **Read the Content Bible** (`docs/strategy/CONTENT-BIBLE.md`), specifically Section 3 (Audience-Specific Messaging) and Section 4 (The Narrative Arsenal).
2. **Read the GTM Plan** (`docs/strategy/GO-TO-MARKET-PLAN.md`), specifically the Positioning section.
3. **Read the Investor Brief** (`docs/strategy/INVESTOR-BRIEF-2026.md`) for market context.

Generate:

### For the specified audience, produce:

1. **One-sentence positioning** — The core value proposition for this specific audience
2. **30-second pitch** — What to say if you have 30 seconds with this person
3. **3 supporting messages** — Priority-ordered proof points that matter to them
4. **Primary narrative** — Which of the 5 Headline Narratives to lead with and why
5. **Key objection + response** — The #1 objection this audience will raise and how to handle it
6. **Recommended analogy** — Which of the 5 approved analogies resonates most
7. **CTA** — What you want this person to do next

### Tone calibration:
- Match the tone adjustment table in Content Bible Section 1
- Use the vocabulary appropriate for this audience's technical level
- Reference specific concerns and priorities for this audience type
