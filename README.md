# Flint Glass

Governance infrastructure for AI-generated UI code.

## Install

```bash
npx flint-glass
```

Or download the latest release for macOS, Linux, or Windows from the [Releases](https://github.com/Jtiem/citadel-flint/releases) page.

## What is Flint?

Flint enforces design systems, accessibility standards (WCAG 2.1 AA), and brand compliance at the AST level — before AI-generated code reaches production.

## Knowledge artifacts

Lessons learned from building Flint, written for designers and founders who are past the "wow, it wrote my function" phase of AI coding and want to ship something real.

- **[The AI Coding Playbook](docs/AI-CODING-PLAYBOOK.md)** — sixteen themes on what actually compounds when you build with AI as a core collaborator. Pull quotes, scenario openers, a designer's glossary, honest open empirical questions, and citations to the prior art where it exists.
- **[The AI Coding Cookbook](docs/AI-CODING-COOKBOOK.md)** — working examples and copy-paste recipes for the practical mechanics referenced throughout the playbook. Mix of one-line stubs (filling in over time) and fully worked recipes (Part 15) covering contract-first feature setup, reviewer agents, hooks, MCP tools, and prompt A/B testing.

Both are evergreen and update independently of binary releases.
